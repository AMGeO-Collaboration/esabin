Defining Bins and Binning Data
------------------------------

esabin.esagrid
==============

.. currentmodule:: esabin.esagrid

The `esagrid` module defines two types of binning scheme.

#. Equal-solid-angle bins (Esagrid)
#. Constant longitude bins (ConstantAzimuthalSpacingGrid)
